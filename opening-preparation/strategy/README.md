## 🎯 StudioJinsei コンセプト

**見えない思いを、見える形に**

**考えが動きに変わるきっかけをつくるスタジオ**

📄 [会社コンセプト詳細](./company-concept.md)

**核心：**
- 形や言葉になっていない思いを、形にする
- 止まっていたことに手を入れ、次に進むための選択肢が見えてくる状態をつくる
- 判断軸：① 構造として残るか ② 迷いが減るか ③ 次の展望がひらくか

📄 [デザインコンセプト詳細](./design-concept.md)

**ビジュアル方向性：**
- シンプル、でも冷たくない
- 整理された、でも堅苦しくない
- プロフェッショナル、でも距離がない
