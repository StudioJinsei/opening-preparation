# StudioJinsei｜Design Concept

## 0. デザインの大前提

### 会社コンセプトとの接続
StudioJinseiは「考えが動きに変わるきっかけをつくるスタジオ」。
デザインもまた、**止まっていた手が動き出す瞬間を演出する**ものでなければならない。

### デザインが果たす役割
1. 考えを整理する「構造」を見せる
2. 次に何をすればいいか「迷わない」状態をつくる
3. 先の展望が「ひらく」感覚を与える

---

## 1. 視覚的な方向性

### キーワード
- **シンプル、でも冷たくない**
- **整理された、でも堅苦しくない**
- **洗練、でもとっつきにくくない**
- **プロフェッショナル、でも距離がない**

### やらないデザイン
- SNSキラキラ系（派手、ポップ、エモ強め）
- スピリチュアル系（ふわふわ、抽象的、神秘的）
- コンサル系（冷たい、堅い、距離がある）
- 低単価チープ感（安っぽい、手抜き感）

---

## 2. ビジュアル要素

### カラーパレット（StudioJinsei公式）

| 名前 | カラーコード | 用途 | 意味 |
|------|-------------|------|------|
| **ソフトミント** | `#A8D5BA` | メインカラー | 止まっていたものが動き出す「はじまり」の色 |
| **ダークティール** | `#1D4E4A` | 見出し・強調 | 構造を示す「軸」の色 |
| **ペールミント** | `#E8F5EE` | 背景・余白 | 次の展望がひらく「余地」の色 |
| **ホワイト** | `#FFFFFF` | ベース | 迷いをなくす「クリア」な状態 |
| **ウォームグレー** | `#4A4A4A` | 本文 | 親しみと信頼のバランス |
| **ライトグレー** | `#F5F5F7` | 背景（Apple風） | 洗練された余白 |

### 配色の原則
- **メイン：ミント系**（動き出す、次が見える）
- **サブ：ティール**（構造、軸）
- **ベース：ホワイト**（迷わない、クリア）
- **アクセント控えめ**（過度な主張をしない）

## 3. キャラクター設定（オフィシャルコトネちゃん）

### 基本設定
| 項目 | 設定 |
|------|------|
| **役割** | StudioJinsei 代表 |
| **立ち位置** | プロフェッショナル、でも親しみやすい |
| **雰囲気** | 穏やかな微笑み、安心感、信頼感 |
| **年齢層** | 20代後半〜30代前半の落ち着き |

### ビジュアル
| 項目 | 設定 |
|------|------|
| **スタイル** | 手書き風キレイめ |
| **線の質感** | 柔らかい、少しラフ、温かみ |
| **色味** | 落ち着いた、洗練、トーン控えめ |
| **髪** | ダークロング、ナチュラル |
| **服装** | ホワイト系、エレガント |
| **アイコン** | ミントグリーンのバケットハット（葉柄） |

### 表情の原則
- 穏やかな微笑み（強すぎない）
- 信頼感を与える（プロとして）
- 親しみやすさ（距離を縮める）
- 「次、動けそう」と思わせる安心感

---

## 4. デザイン原則（Apple風ミニマル）

### 余白の使い方
- **たっぷり取る**：呼吸するデザイン
- **余白 = 展望がひらく余地**
- 詰め込まない、逃げ道を残す

### シンプルの定義
- 要素を絞る（3つまで）
- ノイズを排除（装飾は最小限）
- 一目で理解できる構造

### 洗練の基準
- 高品質（細部までこだわる）
- 一貫性（カラー、フォント、トーンを統一）
- でも、堅苦しくない（温かみを残す）

## 5. タイポグラフィ

### 日本語フォント
| 用途 | フォント | 理由 |
|------|---------|------|
| 見出し | Zen Maru Gothic (Medium/Bold) | 柔らかさと構造のバランス |
| 本文 | Noto Sans JP (Regular) | 読みやすさと洗練 |

### 英語フォント
| 用途 | フォント | 理由 |
|------|---------|------|
| ロゴ・見出し | Poppins (Medium/SemiBold) | モダン、プロフェッショナル |
| 本文 | Inter (Regular) | 可読性が高い |

### フォントサイズ階層
```
H1: 32-40pt（見出し大）
H2: 24-28pt（見出し中）
H3: 18-20pt（見出し小）
本文: 14-16pt（読みやすさ重視）
キャプション: 12pt
```

## 6. ロゴ設計の方向性

### ロゴのコンセプト
- **StudioJinsei**を主体に
- Jinseiは裏テーマ（控えめに）
- 構造がわかる形
- 次に進める感じ（矢印的要素、でも直接的でない）

### ロゴのスタイル候補
1. **シンボル + ワードマーク**
   - シンボル：ミニマルな葉、矢印的な形
   - ワードマーク：Poppins、ミント + ティール

2. **ワードマークのみ**
   - StudioJinsei をシンプルに
   - Studio を細字、Jinsei を太字（構造の対比）

3. **アイコンタイプ**
   - SJ のモノグラム
   - ミニマル、洗練、温かみ

### ロゴで避けること
- 複雑すぎる図形
- 多色使い（2-3色まで）
- キラキラ、スピリチュアル感
- 安っぽいクリップアート感

## 7. サイトデザインの方向性

### トップページ構成案
```
┌───────────────────────────────────┐
│                                   │
│   [ ロゴ ]          [ メニュー ]   │
│                                   │
│                                   │
│   考えが動きに変わる               │
│   きっかけをつくるスタジオ         │
│                                   │
│   [ シンプルなCTA ]                │
│                                   │
└───────────────────────────────────┘

┌───────────────────────────────────┐
│                                   │
│   構造 │ 迷いが減る │ 次の展望    │
│                                   │
│   [ 3つの価値を3カラムで ]         │
│                                   │
└───────────────────────────────────┘

┌───────────────────────────────────┐
│                                   │
│   サービス紹介                    │
│   （シンプルなカード形式）         │
│                                   │
└───────────────────────────────────┘
```

### デザイン要素
- **ヒーローセクション**：ミントグリーンの柔らかいグラデーション
- **セクション区切り**：余白で区切る（線は使わない）
- **カード**：角丸、影は薄く
- **CTA**：ミント背景、ティールテキスト、シンプルボタン

### ページ構成
| ページ | 目的 |
|--------|------|
| トップ | コンセプト伝達、次のアクションへ |
| サービス | 何ができるか、明確に構造化 |
| 会社概要 | 信頼感、でもストーリーは語らない |
| お問い合わせ | シンプルなフォーム、迷わせない |

## 9. 判断軸（デザイン版）

すべてのデザイン要素は以下の3つで判断する。

1. **構造として見えるか**
   → ごちゃごちゃしていないか、整理されているか

2. **迷いが減るか**
   → 次に何をすればいいかが見えるか

3. **次の展望がひらくか**
   → 余白、余地、可能性を感じるか

ひとつでも欠けるなら、StudioJinseiのデザインではない。
